Rifana Project



When darkness is taking over. The stars are dying. Help this project to succeed and revive the dying stars. Move your space ship. Avoid Enemies. Shoot a starlight bullet into a dying star, and revive it.

#revivethestars #notaspaceshootergame







This game was planned to enter a competition. But was cancelled for some reasons.

Rifana Project is the most complex game I ever created yet.
Features:
- Virtual-Analog Joystick Control
- Select Characters
- Option Menu (Sound, Sensitivity, etc)
- Tutorial
- Multiple Game mode (Arcade, Survive, Story(Coming Soon))
- Change Joystick Mode (Left/Right Hand)
- Explosion and Shaking Effects
- Glow in the dark Art.
